# v2ray-test-config
Free V2Ray config for community testing
# Community Feedback Welcome

> **Disclaimer**: This is a free test service. Not for critical traffic or illegal activity. Configs may change or expire at any time.

## 🧪 How to test & give feedback

1. Add the subscription link to your client.
2. Test for at least 24–48 hours.
3. Reply in **Issues** or **Discussions** with:
   - Your approximate location (country/city)
   - Your ISP (e.g. Vodafone, Rostelecom, Deutsche Telekom)
   - Average speed / latency
   - Any disconnects or slowdowns (with approximate time)
   - Which client you use (version too, if possible)

## 🤝 Why am I doing this?

I want to:
- Improve config quality based on real-world data
- Understand which protocols work best in different regions
- Provide a truly usable free option for testing
